# JEEPilot Study Vault

An all-in-one study management platform designed for JEE aspirants to organize notes, track progress, manage subjects, and streamline preparation.

## Live Demo

🌐 [https://jeepilot-study-vault.onrender.com/](https://jeepilot-study-vault.onrender.com/)

---

# Features

* 📚 Subject-wise study organization
* ✅ Topic and chapter tracking
* 🧠 JEE-focused preparation workflow
* 📈 Progress management system
* 📝 Notes and study material storage
* 🌙 Modern responsive UI
* 💾 Persistent local/session data handling
* ⚡ Fast and lightweight performance

# Installation

## Clone the Repository

```bash
git clone https://github.com/your-username/jeepilot-study-vault.git
cd jeepilot-study-vault
```

## Install Dependencies

```bash
npm install
```

## Start the Development Server

```bash
npm run dev
```

or

```bash
npm start
```

---

# Environment Variables

Create a `.env` file in the root directory and add the required environment variables.

Example:

DATABASE_URL=your_database_url
